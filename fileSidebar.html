For guided setup select a "home" folder for your purchase system to live in, then continue below.

Fill out the parameters below and click "Build Purchase Tracker". You will need a Google Sheets file representing a blank purchase order. MAE projects may copy the file here.

This section controls generation of a purchase order form.

Check how many lines appear on your department's PO form. The following information will be requested in the form:
Vendor, Vendor Contact Phone, Website, Shipping, Requestor Name, Requestor Phone, Special Instructions.
The following information will be requested for each line item:
Quantity, Unit of Measure, Description, Item #, Price.

If your form requires additional information, please add it manually after generating the file.

You need to set up the CurrentPO sheet so information populates in the correct places.
To make this easy, we generated some named ranges corresponding to the relevant fields. (You can also define your own!)
For example, to display the current PO vendor name in a cell, enter the formula =INDIRECT(vendorName)
A list of all named fields will appear in the sheet.
